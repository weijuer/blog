# 介绍

本小站将着眼于开源的思想，本厮会将自己在实际工作和学习新技术同时，利用闲暇时间记录本厮成长的点滴，尽量还原开发实践当中遇到的常见问题以及解决途径或者思路分享给大家。同时会将不定时更新小案例代码，以便小白们学习，大神们请绕道，谢谢！

## Why weijuer?

“Weijuer”不仅仅是一个名字，更是一种代码精神，追求新、尝试奇，喜欢被追逐的快感，“我只是比你来坑的早，相信我你也可以！”
